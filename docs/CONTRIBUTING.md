# Contributing to ComputeImpact
Thank you for your interest in contributing! ComputeImpact is an early-stage project, and your ideas, bug reports, and pull requests are welcome. I'm looking forward to building something amazing with you! 

## Ways to Contribute
- Report bugs
- Suggest features
- Improve documentation
- Build UI components
- Help shape the roadmap

## Development Setup
### Backend: Node.js
```
cd backend
npm install
npm run dev
```
### Frontend: React + Vite
```
cd frontend
npm install
npm run dev
```
## Pull Request Guidelines
- Keep PRs focused and scoped
- Include clear descriptions
- Add tests if/when possible
- Update documentation if needed
